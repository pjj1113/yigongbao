# 首页看板 前端接收数据文档

**接口地址**: `GET /dashboard/{roleCode}`  
**前端类型定义**: `src/api/dashboard.ts`

---

## 请求参数

```ts
{
  timeRange: 'today' | 'week' | 'month' | 'quarter' | 'year' | 'custom'
  startTime?: string  // custom 时必填，格式 "2026-05-01 00:00:00"
  endTime?: string    // custom 时必填
}
```

---

## 响应结构

```ts
// 顶层
interface DashboardVO {
  cards: CardVO[]          // 数据卡片，
  charts: ChartVO[]        // 图表，
  todos: TodoVO[]          // 待办事项
  system?: SystemVO        // 系统监控，仅 super_admin 返回
}

interface SystemVO {
  healthStatus: 'healthy' | 'warning' | 'error'  // 系统健康状态
  avgResponseTime: string                          // 平均响应时间，如 "120ms"
  onlineUsers: number                              // 当前在线用户数
  avgOrderCycle: string                            // 平均订单周期，如 "432h"
}
```

### CardVO — 数据卡片

```ts
interface CardVO {
  key: string                      // 唯一标识，如 "totalOrders"
  title: string                    // 卡片标题，如 "我的订单总数"
  value: number | string           // 主数值，如 156 或 "92.3%"
  unit?: string                    // 单位，如 "单" "元" "天"，可选
  change?: string                  // 环比文字，如 "+12.5%" "-3单"
  trend?: 'up' | 'down' | 'flat'  // 趋势方向（up=红 down=绿）
}
```

**示例**:
```json
{
  "key": "totalOrders",
  "title": "我的订单总数",
  "value": 156,
  "unit": "单",
  "change": "+12.5%",
  "trend": "up"
}
```

---

### ChartVO — 图表

```ts
interface ChartVO {
  key: string                                    // 唯一标识
  title: string                                  // 图表标题
  type: 'line' | 'bar' | 'pie' | 'funnel' | 'ring'
  data: LineChartData | BarChartData | PieChartData | FunnelChartData
}
```

#### 折线图 (type: 'line')

```ts
interface LineChartData {
  xAxis: string[]          // X轴标签，如 ["5/12", "5/13", ...]
  series: {
    name: string           // 系列名，如 "新建订单"
    data: number[]         // 与 xAxis 等长
  }[]
}
```

**示例**:
```json
{
  "key": "orderTrend",
  "title": "订单趋势",
  "type": "line",
  "data": {
    "xAxis": ["5/21", "5/22", "5/23", "5/24", "5/25", "5/26", "5/27"],
    "series": [
      { "name": "新建订单", "data": [12, 15, 18, 14, 20, 16, 22] },
      { "name": "完成订单", "data": [8, 10, 12, 11, 15, 13, 18] },
      { "name": "驳回订单", "data": [1, 2, 1, 0, 2, 1, 1] }
    ]
  }
}
```

#### 柱状图 (type: 'bar')

```ts
interface BarChartData {
  xAxis: string[]          // X轴标签，如人名、部门名
  series: {
    name: string
    data: number[]
  }[]
}
```

**示例**:
```json
{
  "key": "workerLoad",
  "title": "生产员工作负载",
  "type": "bar",
  "data": {
    "xAxis": ["张三", "李四", "王五", "赵六"],
    "series": [
      { "name": "生产中", "data": [10, 12, 11, 13] },
      { "name": "已完成", "data": [32, 28, 35, 30] }
    ]
  }
}
```

#### 饼图 (type: 'pie') / 环形图 (type: 'ring')

```ts
interface PieChartData {
  items: {
    name: string
    value: number
  }[]
}
```

**示例**:
```json
{
  "key": "phaseDistribution",
  "title": "订单阶段分布",

```

interface FunnelChartData {

### TodoVO — 待办事项

```ts
interface TodoVO {
  id: number
  title: string       // 待办描述，如 "待审核订单"
  count?: number      // 数量，可选
  link?: string       // 点击跳转路由，可选
  urgent?: boolean    // 是否加急，可选
}
```

**示例**:
```json
{ "id": 1, "title": "待审核订单", "count": 5, "link": "/order?status=pending", "urgent": false }
```

## 扩展字段说明

部分卡片需要展示子数据（如今日已质检的通过数/返工数），约定在 `value` 中用 `/` 分隔，或后端额外返回 `subValue` 字段（前端 `CardVO` 可扩展）：

```json
{
  "key": "todayQcDone",
  "title": "今日已质检",
  "value": 23,
  "change": "通过 20 / 返工 3"
}
```

即：`change` 字段不仅用于环比，也可用于补充说明文字（无 `trend` 时不显示箭头图标）。
